SENAI

Anatomia CSS - Seletores

Por tag:

  • Ex.: input corresponderá a qualquer elemento 'input'>;

Por classe:

  • Ex.: .index irá corresponder a qualquer elemento que tenha o índice de classe “index”;

Por ID:

  • Ex.: #toc irá corresponder ao elemento que possuir o id = “toc”;

Por Atributo:

  • Ex.: input [type = “email”] {} corresponderá a todo elemento que tiver “email” como atributo;
