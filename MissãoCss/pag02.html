O que é o CSS?

CSS é a sigla para Cascading Style Sheets, que em português significa Folha de Estilo em Cascata. Trata-se de uma linguagem de estilo usada para definir a apresentação e o visual de documentos escritos em linguagens de marcação, como HTML ou XML.

Na pratica, sua principal função é harmonizar os elementos visuais inseridos pelo HTML, tornando o produto final esteticamente agradável.
