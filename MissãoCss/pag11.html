Anatomia CSS - Propriedades

Propriedades são so aspectos do HTML os quais vão ser estilizados. Ex.:

  • Color;
  • Font-size;
  • Background-color;
  • Etc.

Valores

São os valores aos quais você atribui uma propriedade, para que as alterações tomem efeito. Ex.:

  • Códigos hex;
  • Unidades de medida para tamanho de elementos (pixels, porcentagem para tamanho relativo);
