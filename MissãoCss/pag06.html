Como ligar o CSS ao HTML?

Meios de “linkar” o css com o html e qual é mais usado no mercado

Inline (Embutido): CSS é aplicado diretamente no elemento.

  
            <p style = color: pink; font-size: 16px;> </p>

Este é um parágrafo com estilo inline.

Externo: CSS fica em um arquivo nome.CSS e você “linka” ele com esse arquivo através da TAG link. 


            <head>

            < link rel="stylesheet" href="style.css">

            <head>

Interna: CSS é embutido no head através da tag Style.


         <!DOCTYPE html>
            <html>
                <head>
                   <br>
                    <style>
                        h1 { color: green; }
                    <style>
                   </head>

Este é um título com estilo imbutido no head.
